A list of all my public projects, technical or technical-adjacent. For a more throrough history of all my projects, take a look at [my github](https://github.com/somaniarushi).


## The List

**BEGIN**: Primary developer of [BEGIN @ Berkeley](https://begin.berkeley.edu). Made in Gatsby.

**Narcissus**: To begin with, the hub for everything else I make, this website! Developed, designed and maintained in Hugo. Fork it [here](https://github.com/somaniarushi/me).

**Carnot**: Developing my own browser engine. A toy version for how Chrome works. Fork it [here](https://github.com/somaniarushi/carnot).

**Dungeons**: Web-socket enabled online RPG combat platform. A simpler version of Roll20, made for players with some programming knowledge. Fork it [here](https://github.com/somaniarushi/dungeons).

**Respond**: Cloning React to learn how it works (And maybe even improve it). Fork it [here](https://github.com/somaniarushi/repond).

**Tempest**: Dynamic thought-tree generator, for those trying to bring order to their mental gymnastics, made in React and hosted locally. Fork it [here](https://github.com/somaniarushi/tempest).

**Prometheus**: Quick character sheet generator for RPG games — for beginners! Made in React. A simpler [fastcharacter.com](https://fastcharacter.com). Fork it [here](https://github.com/somaniarushi/prometheus).

**RISCer**: Hexadecimal input to RISC instruction converter, made for future 61C students. Terminal-based, developed in Python! Fork it [here](https://github.com/somaniarushi/riscer).

**Hullo**: A real-time internal relay chat (IRC) app, made using socket.io and Node.js. Play around at [hullo.amks.me](https://hullo.amks.me). Fork it [here](https://github.com/somaniarushi/hullo)

**Clippy**: A webpage that implements a clipboard. Made as a fun thought experiment. Try it out at [clippy.amks.me](https://clippy.amks.me) or read about it [here](../blog/clipboard.md). Fork it [here](https://github.com/somaniarushi/clippy).

**Jump**: Chrome extension for storing shortlinks. Currently highly personalized, made in JS. Fork it [here](https://github.com/somaniarushi/jump).

**Gitlet**: Creating a copy of Git, implementing most of its main features. Made for CS61B at UC Berkeley. Not made publicly available by request of CS61B course-staff.

**Swallowtail**: Customizable dark mode toggle for React components. Made in React. Available for use on npm [here](https://www.npmjs.com/package/@asomani/darkmode). Fork it [here](https://github.com/somaniarushi/swallowtail).

**VATT**: Video-to-video automatic dubbing and subtitling tool, to make any video on the internet available to everyone, regardless of language barriers. Made in Python using Google, Amazon and IBM text-to-speech APIs. Fork it [here](https://github.com/somaniarushi/vatt).

**Tortoise**: Digital art made using Python Turtles! Extension of CS61A's Scheme Art Contest. Fork it [here](https://github.com/somaniarushi/tortoise).

**Tesseract**: A 4-dimensional cube, rendered in Virtual reality for Google Cardboards and Oculus VR. Made in Unity. Fork it [here](https://github.com/somanairushi/tesseract).

**Cannonball**: Implementation of graph theory shortest path implementations for massive graphs. Made for CS170 at UC Berkeley, with [@itsvs](https://github.com/itsvs). Fork it [here](https://github.com/itsvs/proj170).
