Using Obscure CSS Notation For Dark Mode

Jun 16 2020

I’ve written a more thorough take in react here, which I recommend over this, but this one’s too interesting to leave out.

Fundamental Idea: use the CSS + selector to match sibling items with a checkbox label.

First, let’s set up the HTML file, importing a CSS file we’ll set up later.

<html>
    <head>
        <title>
            CS61A | Introduction
        </title>
        <link rel="stylesheet" href=""> ** Set Up Later **
    </head> 
</html>

Then, let’s add a checkbox in the body as our toggle.

<html>
    <head> ... </head>
    <body>
        <input type="checkbox" id="cb" />
    </body>
</html>

Congratulations! We have a toggle that does nothing!

Never fear, let’s march on ahead. Now let’s set up a division we’ll call wrapper and a label for checkbox inside of it. This label is what we’ll use to design our toggle.

<html>
   <head> ... </head>
   <body>
       <input type="checkbox" id="cb" />

       <div id="wrapper">
           <label id="toggler" for="cb">
           </label>
       </div>
   </body>
</html>

Now, we’ll add two spans into the label div –– one for the toggle outline, and the other for the toggle indicator.

<html>
  <head> ... </head>
  <body>
      <input type="checkbox" id="cb" />

      <div id="wrapper">
          <label id="toggler" for="cb">
               <span class="toggle-border">
                  <span class="toggle-indicator"></span>
              </span>
          </label>
      </div>
  </body>
</html>

Quite boring till now. No worries! CSS is where the magic happens.

Let’s set up a css file now, and link it in the header of the HTML file that we’d left blank earlier. (Don’t forget!) Then, let’s begin the css file by removing the standard checkbox.

#cb {
    display: none;
}

Now let’s set up the actual toggle.

  #toggler .toggle-border {
    border: 1px solid #1e1f26;
    width: 50px;
    height: 20px;
    display: block;
    border-radius: 20px;
    position: relative;
  }

  #toggler .toggle-indicator {
    width: 16px;
    height: 16px;
    border-radius: 50%;
    position: absolute;
    background-color: #1e1f26;
    top: 2px;
    left: 2px;
    transition: all 0.3s ease-in-out;
  }

Very nice! We’re setting up a curved border .toggle-border with a block display to prevent collapse. Then we set up a .toggle-indicator circle and position it properly. The position: relative is important to make the position: absolute in toggle-indicator possible and position it comfortably.
